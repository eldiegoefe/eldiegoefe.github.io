Los directorios típicos son los siguientes, aunque hay variaciones en las distintas distribuciones de linux: [/bin] ejecutables (Essential commands that everyone needs to use at any time) [/usr] A complex hierarchy of additional programs and files. [/usr/bin] normal user executables. The commands that aren’t essential for users but are useful [/sbin] (superuser core executables) The commands the system Administrator needs access to.
